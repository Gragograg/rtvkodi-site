Image Loading Help

If your images are not loading, here are the most common solutions:

1Check image file names

Make sure your image files are named exactly as referenced in the HTML:

  • Screenshot-2025-09-21-at-12.11.39.png
  • Screenshot-2025-09-21-at-12.09.17.png
  • Screenshot-2025-09-21-at-12.09.55.png
  • Screenshot-2025-09-21-at-12.10.53.png
  • Screenshot-2025-09-21-at-12.08.16.png
  • Screenshot-2025-09-21-at-12.06.13.png

Filenames are case-sensitive on GitHub. Check for exact matches.

2Check folder structure

Your repository should have this structure:

  • index.html (this file)
  • img/ (folder)
    • Screenshot-2025-09-21-at-12.11.39.png
    • Screenshot-2025-09-21-at-12.09.17.png
    • Screenshot-2025-09-21-at-12.09.55.png
    • Screenshot-2025-09-21-at-12.10.53.png
    • Screenshot-2025-09-21-at-12.08.16.png
    • Screenshot-2025-09-21-at-12.06.13.png

3Check GitHub Pages settings

Make sure GitHub Pages is configured correctly:

  • Go to your repository Settings → Pages
  • Set "Source" to "Deploy from a branch"
  • Set "Branch" to "main" (or "master") and folder to "/ (root)"
  • Wait a few minutes for the site to deploy

4Alternative solution

If you're still having issues, you can:

  • Upload your images to a service like Imgur
  • Use the direct image links in your HTML
  • Replace img/filename.png with the full URL
